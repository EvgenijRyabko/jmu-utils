<h1>JMU Utils npm package</h1>


This is a npm package for LGPU microservice projects.


<details>
  <summary>Current package utils list:</summary>
  <br />

  <ul>
    <li><b>Request parser</b> for nest.js microservice (Transform string request to a typed obj of express Request type).</li>
    <li><b>params-meta decorator</b> (A extension for request parser to set params template metadata. Needed to correctly parse params).</li>
  </ul>
</details>
